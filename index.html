Simple Text Positioning

Using semantic tags, I've laid out a very basic website. I'm styling the position of each section.

I've worked in IT Quality for years, but now I am working on building websites. As of September, 2021, I'm still pretty much a newbie.

So this is my first website where I have set it up and laid out the positioning of each element with almost no text formatting (ok, so obviously there's a little text formatting floating around somewhere, lol). It does not rely on external frameworks, and it uses only floats to position the text.

I decided to apply this old format as a learning exercise to teach myself how blocks behave on a webpage. In my next website, I will use flexbox to position the text, and I will make the navbar links functional.

For now, take a look and tell me what you think.
